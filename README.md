# PUMPTANK

**Parody *Shark Tank* tribute tokens on [pump.fun](https://pump.fun) (Solana).**

🌐 **Live site: [thepumptank.fun](https://thepumptank.fun)**

PUMPTANK mints an unofficial tribute meme token for **every** *Shark Tank US* product,
~1,481 in all. Each one launches as a **create-only** coin (no insider dev-buys). The ~566
that **walked away without a deal** get the spotlight with a vermilion **"NO DEAL"** card;
the ~915 that *did* land a deal get a neutral tribute card. Every coin feeds the
`$PUMPTANK` index token, which launches **first** with a one-time **10% dev-buy** on the
house wallet and earns a share of fees from across the index.

> Tank* / ABC / Sony. Nothing here is financial advice, and no token carries any promise of

## How it works

| Tier | Count | What it gets |
|------|-------|--------------|
| **No-deal** | ~566 | Create-only launch + vermilion "NO DEAL" card |
| **Got a deal** | ~915 | Create-only launch + neutral tribute card (no "NO DEAL" badge) |
| **`$PUMPTANK` index** | 1 | Launched first; `create_v2` + a one-time **10% dev-buy** (via a reusable Address Lookup Table) on the house wallet |

Creator fees route entirely on-chain (Pump Fees V2): un-opted coins pool to the house;
on a founder's opt-in, a coin locks to an **80/10/10** split (creator / main-token /
referrer).

## Index economics (`$PUMPTANK`)

`$PUMPTANK` is the index token. It sits above the ~1,481 tribute coins and is tied to a
slice of the activity across all of them.

Here's how the fees flow up. Every tribute coin earns creator fees on pump.fun when it
trades, and those fees route on-chain:

- Before a founder opts in, that coin's creator fees pool to the project's house treasury.
- After a founder opts in, the coin locks to an on-chain 80 / 10 / 10 split: 80% to the
  founder, 10% to the `$PUMPTANK` treasury, 10% to whoever onboarded them. With no
  onboarder it's 80 / 20 (founder / treasury).
- The `$PUMPTANK` coin's own trading fees go to the treasury too.

So as the catalog trades and more founders opt in, the `$PUMPTANK` treasury builds up real,
on-chain fee revenue from across the whole index.

What the treasury *can* go toward: things like marketing, growth, deeper on-chain
liquidity, and the roadmap below. That's an option, not a promise. Nothing here commits the
team to any buyback, burn, distribution, or outcome, and no token carries any promise of
value or return. See the disclaimer.

## Where this could go (not a promise)

The catalog is step one. Past *Shark Tank*, we'd like to open this up to anyone with an
idea: people who pitch us directly. Down the road the `$PUMPTANK` treasury could be used to
spotlight or back products the community believes in, maybe by minting a tribute coin for an
accepted pitch and routing fees back to that founder, the way the catalog works today.

This is just where we'd like to take it, not a commitment. Any of it might change, get
delayed, or never happen. None of it is financial advice, an offer, or a guarantee.

## Repo layout

| Path | Stack | Role |
|------|-------|------|
| `pipeline/` | Python (pandas, pydantic, Pillow) | Ingest the *Shark Tank* dataset, mark every product `include` (launched) plus its real `got_deal` outcome, generate token text + a card for all ~1,481. Output: `data/products.json` + `data/token_images/`. |
| `launcher/` | TypeScript (`@solana/web3.js`, `@pump-fun/pump-sdk`) | Mint every product coin **create-only** (`npm run launch`); launch the `$PUMPTANK` index token with a 10% dev-buy (`npm run launch:index`: `create_v2` + atomic buy via a reusable Address Lookup Table). On-chain fee CLI. |
| `web/` | Next.js (App Router) | Public gallery of the tokens + a founder opt-in / verification page. |
| `data/` | JSON + PNG | `products.json` (all ~1,481) and `token_images/`. Regenerated by the pipeline, not hand-edited. |
| `docs/` | Markdown | Specs, plans, runbooks, licenses, curation audit reports. |
| `scripts/` | Node / Python | YouTube-link merge, Tally form builder, web asset copy. |

### Data model (`data/products.json`)
- `include: true`: **launched** (every product; all create-only).
- `got_deal`: real on-air outcome. Drives card styling ("NO DEAL" badge vs neutral) and the site's deal/no-deal treatment.
- `selection.rank`: editorial rank among the no-deal pitches (reach/ambition/findability), used only to order the website. **No product gets a dev-buy** (that's reserved for the `$PUMPTANK` index token).

Two web-verified override maps in `pipeline/pumptank_pipeline/config.py` correct the
source data (its `Got Deal` column is unreliable):
- **`GOT_DEAL_OVERRIDES`**: fixes wrong deal flags (e.g. Packback closed a deal on air).
- **`EXCLUDE_IDS`**: drops too-big/exited companies from the editorial ranking (they still
  launch create-only like everything else). See `docs/no-deal-audit-2026-06-04.md`,
  `docs/too-big-sweep-2026-06-04.md`.

## Running it

### Pipeline (regenerate data + cards)
The source dataset isn't committed (see *Data & licensing*). Download it to
`data/raw/Shark Tank US dataset.csv`, then:
```bash
cd pipeline
pip install -e ".[dev]"
python -c 'from pumptank_pipeline import cli, config; cli.run(config.RAW_DIR/"Shark Tank US dataset.csv", config.DEFAULT_OUTPUT, config.DEFAULT_SCHEMA)'
pytest -q
```

### Web (local dev)
```bash
cd web && npm install && npm run dev   # copies token images, then starts Next.js
```

### Launcher
```bash
cd launcher && npm install && npm run build && npm test
# launching/fees need a funded wallet + RPC (see Configuration)
```

## Configuration

**No secrets live in this repo.** Keys and wallets are supplied at runtime via environment
variables and `.gitignore`d local files:

| Var | Used by | Notes |
|-----|---------|-------|
| `WALLET` | launcher | House wallet **secret key as a JSON byte array**. Keep it secret, never commit. |
| `RPC_URL` | launcher | Solana RPC endpoint (your own key). |
| `PRIORITY_FEE`, `SLIPPAGE_BPS`, `MAX_TOTAL_SPEND_SOL` | launcher | Launch tuning. |
| `INDEX_DEV_BUY_SOL` | launcher | `$PUMPTANK` index dev-buy SOL cap (`npm run launch:index`; default 3.5). |
| `TALLY_API_KEY`, `TALLY_WORKSPACE_ID` | `scripts/create-tally-form.mjs` | Founder-verification form. |
| `SITE_URL` | web | Canonical site URL for metadata. |

Intentionally git-ignored secret/local files: `.env*`, `*.key`, `*.pem`,
`data/.mint-keys/`, `data/fee-config.json`, `data/launch-alt.json`.
