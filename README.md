# PUMPTANK

**Parody *Shark Tank* tribute tokens on [pump.fun](https://pump.fun) (Solana).**

PUMPTANK mints an unofficial tribute meme token for **every** *Shark Tank US* product —
~1,481 in all. The companies that **walked away without a deal** get the spotlight: a
vermilion **"NO DEAL"** card, and the curated **top 100** of them get a small on-chain
dev-buy ("got no deal on the show? here's a little something — tokens"). The ~915 that
*did* land a deal launch too, as create-only tribute coins. Every coin feeds the
`$PUMPTANK` main token.

> ⚠️ **Disclaimer.** These are **unofficial fan tribute & parody tokens**. They are **not
> affiliated with, endorsed by, or sponsored by** the companies, their founders, or *Shark
> Tank* / ABC / Sony. Nothing here is financial advice, and no token carries any promise of
> value. Company names and likenesses are used for identification/commentary only; a
> trademark & likeness review is recommended before any public launch.

## How it works

| Tier | Count | What it gets |
|------|-------|--------------|
| **Top-100 no-deal** (curated) | 100 | Full launch + **1.5% dev-buy** + "NO DEAL" card |
| **Other no-deal** | ~466 | Create-only launch + "NO DEAL" card |
| **Got a deal** | ~915 | Create-only launch + generic tribute card (no "NO DEAL" badge) |

Creator fees route entirely on-chain (Pump Fees V2): un-opted coins pool to the house;
on a founder's opt-in, a coin locks to an **80/10/10** split (creator / main-token /
referrer).

## Repo layout

| Path | Stack | Role |
|------|-------|------|
| `pipeline/` | Python (pandas, pydantic, Pillow) | Ingest the *Shark Tank* dataset → mark every product `include` (launched), select the top-100 no-deal for `dev_buy` → generate token text + a card for all ~1,481. Output: `data/products.json` + `data/token_images/`. |
| `launcher/` | TypeScript (`@solana/web3.js`, `@pump-fun/pump-sdk`) | Mint each coin: **dev-buy** path (`create_v2` + atomic buy via a reusable Address Lookup Table) for the 100, **create-only** for the rest. On-chain fee CLI. |
| `web/` | Next.js (App Router) | Public gallery of the tokens + a founder opt-in / verification page. |
| `data/` | JSON + PNG | `products.json` (all ~1,481) and `token_images/`. **Regenerated by the pipeline — not hand-edited.** |
| `docs/` | Markdown | Specs, plans, runbooks, licenses, curation audit reports. |
| `scripts/` | Node / Python | YouTube-link merge, Tally form builder, web asset copy. |

### Data model (`data/products.json`)
- `include: true` — **launched** (every product).
- `dev_buy: true` — gets the 1.5% dev-buy (the curated top-100 no-deal).
- `outcome.got_deal` — real on-air outcome; drives card styling (NO DEAL badge vs generic).
- `selection.rank` / `selected` — rank within the no-deal dev-buy pool.

The top-100 dev-buy set is curated via two override maps in
`pipeline/pumptank_pipeline/config.py`, both web-verified (the source data's `Got Deal`
column is unreliable):
- **`GOT_DEAL_OVERRIDES`** — fixes wrong deal flags (e.g. Packback closed a deal on air).
- **`EXCLUDE_IDS`** — keeps too-big/exited companies *out of the dev-buy pool* (they still
  launch create-only). See `docs/no-deal-audit-2026-06-04.md`,
  `docs/too-big-sweep-2026-06-04.md`, `docs/launch-set-100.md`.

## Running it

### Pipeline (regenerate data + cards)
The source dataset isn't committed (see *Data & licensing*). Download it to
`data/raw/Shark Tank US dataset.csv`, then:
```bash
cd pipeline
pip install -e ".[dev]"
python -c 'from pumptank_pipeline import cli, config; cli.run(config.RAW_DIR/"Shark Tank US dataset.csv", config.DEFAULT_OUTPUT, config.DEFAULT_SCHEMA)'
pytest -q
```

### Web (local dev)
```bash
cd web && npm install && npm run dev   # copies token images, then starts Next.js
```

### Launcher
```bash
cd launcher && npm install && npm run build && npm test
# launching/fees need a funded wallet + RPC — see Configuration
```

## Configuration

**No secrets live in this repo.** Keys and wallets are supplied at runtime via environment
variables and `.gitignore`d local files:

| Var | Used by | Notes |
|-----|---------|-------|
| `WALLET` | launcher | House wallet **secret key as a JSON byte array** — keep secret, never commit. |
| `RPC_URL` | launcher | Solana RPC endpoint (your own key). |
| `PRIORITY_FEE`, `SLIPPAGE_BPS`, `DEV_BUY_SOL`, `MAX_TOTAL_SPEND_SOL` | launcher | Launch tuning. |
| `TALLY_API_KEY`, `TALLY_WORKSPACE_ID` | `scripts/create-tally-form.mjs` | Founder-verification form. |
| `SITE_URL` | web | Canonical site URL for metadata. |

Intentionally git-ignored secret/local files: `.env*`, `*.key`, `*.pem`,
`data/.mint-keys/`, `data/fee-config.json`, `data/launch-alt.json`.
